‚ö†Ô∏èTroubleshooting

Script doesn't appear in the Scripts list

Solution:

  • Verify the JAR file is in the correct Scripts folder
  • Restart the DreamBot client completely
  • Click the Refresh button in the Scripts tab
  • Check that the JAR file name is exactly: iKingSnipe-Manus-all.jar

Script crashes on startup

Solution:

  • Check the DreamBot console for error messages
  • Ensure you have Java 8 or higher installed
  • Try re-downloading the JAR file from GitHub
  • Verify no other scripts are conflicting

Discord C2 not connecting

Solution:

  • Verify the DISCORD_BOT_TOKEN environment variable is set correctly
  • Ensure your Discord bot has the correct permissions
  • Check that your bot is invited to the correct Discord server
  • Restart DreamBot after setting the environment variable
